## Reference table — field names + cross-skill mappings

These are **verified against the live schemas + working intent files** (not the spec text — which had two field-name errors that this plan corrects):

| Concept | Field path in earthing intent-out.json | Field path in fault-level intent-out.json |
|---|---|---|
| System earthing type | `system_type` (top-level string `"TN-S"` etc.) | n/a |
| Supply Ze | `ze_declared_ohm` (top-level number) | n/a |
| Supply bond type | `supply_bond_type` (top-level string) | n/a |
| Peak PFC at intake | n/a | `fault_currents[]` entry where `node_kind == "transformer_secondary"` → `ifault_ka_max` |
| Peak PFC at MSB | n/a | `fault_currents[]` entry where `node_kind == "board_incoming"` AND board is the MSB → `ifault_ka_max` |
| Top-level wrapper fields | `intent_type`, `intent_version`, `produced_by_skill_version` (schema permits) | `project_id`, `intent_version` (schema permits; `intent_type` NOT permitted) |
| Required fields | `jurisdiction`, `earthing_system` (schema)/`system_type` (live), `circuits`, … | `project_id`, `source_summary`, `fault_currents` |

**Cross-skill consistency rules (INV-11 enforcement):**
- SLD `supply_origin.system_type` MUST equal earthing intent's `system_type` (top-level field — NOT `earthing_system.system_type` as the spec text incorrectly stated)
- SLD `system_metrics.peak_pfc_ka` MUST be within ±0.5 kA of the fault-level intent's `fault_currents[]` entry where `node_kind == "transformer_secondary"` → `ifault_ka_max`

**Schema deviation noted (out of scope for this sprint):** earthing-intent.schema.json declares `earthing_system` (enum) at top level, but the 4 shipped earthing intent-out.json files use `system_type` (different field name). The plan uses the **live intent file** field name (`system_type`) because that's what the 4 working examples use and what SLD INV-11 must cross-check. Schema reconciliation is deferred to a v1.5 cleanup (out of scope; tracked in [[earthing-deferred-followups-queue]] memory).

  - TX-1 LV side: `Ik"max = (c × U_n × 1000) / (√3 × Z_total)` with `c=1.05`, `U_n=400V`, `Z_total = Z_Q_referred + Z_TX = ~5 mΩ` → `Ik"max ≈ 24.0 kA`.   - SDB-L2 (55m): Ik" drops to ~5.2 kA
