# Engineering Philosophy — lighting-layout

1. **Show working first.** Every calculation step shown in chat before JSON emitted.
2. **Never assume, always flag.** UF, MF, LLMF assumptions always tagged.
3. **Round up, never down.** Fixture count always rounded up.
4. **Standards-cited values only.** Every lux target, UGR limit, and MF references its clause.
5. **Compliance by default.** Non-compliant inputs get flagged, not silently accepted.
