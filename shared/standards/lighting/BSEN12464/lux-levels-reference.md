# Maintained Illuminance Targets — BS EN 12464-1:2021

Reference table for the lighting-layout skill. Source: BS EN 12464-1:2021 Table 5.3.

Engineers must verify against the current edition of the standard.
These values are minimums — designers may specify higher levels.
