# BS 5536:1978 — UK Reading of ISO 128 Line Conventions

UK adaptation of ISO 128 line conventions for technical drawings. Largely aligned with ISO 128 line width groups, with UK-specific guidance for electrical schematic conventions.

## Files in this folder

| File | Purpose |
|---|---|
| `meta.json` | Publication metadata, edition, scope |
| `line-conventions.json` | UK line type/width mapping for visible/hidden/centreline/dimension/hatching/border + UK-specific electrical notes |
| `cross-references.json` | Links to ISO 128 (international base) + BS 1192 (UK CDE/layer naming) |

## Cross-references

- **ISO 128** — international base standard for line widths + types
- **BS 1192** — UK information management + layer naming

## Applies to jurisdictions

- **GB** — primary standard (UK electrical drawing convention)
