# ISO 128 — Technical Drawings: General Principles of Presentation

International standard for line widths + line types in technical drawings (multi-part).

## Files in this folder

| File | Purpose |
|---|---|
| `meta.json` | Publication metadata (Parts 20, 21, 23), edition, scope |
| `line-widths.json` | Part 20 §5 — 9 width groups (0.13-2.00 mm) with typical use cases |
| `line-types.json` | Parts 21+23 — 6 line types (continuous, dashed, dot-dash, etc.) |

## Engine-lookup keys

- `line-widths.engine_lookup.by_use["dimension_line"]` → 0.18
- `line-widths.engine_lookup.by_use["border"]` → 1.00
- `line-types.engine_lookup.by_use["centreline"]` → "dot-dash"

- **GB / KE / INT / EU** — primary standard
