# BS 1192 / ISO 19650 — Key Terminology

**Source:** BS 1192:2007+A2:2016 §3 (Definitions); aligned with ISO 19650-1:2018 §3 (Terms and definitions).

## Information Management

| Term | Definition |
|---|---|
| **WIP** | Work in Progress — pre-shared workspace for each individual task team |
| **Shared** | Reviewed and approved information shared between task teams |
| **Published** | Authorized for use by all project parties for the project's intended purpose |
| **Archive** | Documents superseded by newer versions; retained for record |
| **CDE** | Common Data Environment — the agreed source of information for the project, used to collect, manage and disseminate project information |

## Information Requirements

| Term | Definition |
|---|---|
| **EIR** | Exchange Information Requirements — pre-appointment requirements set by the appointing party |
| **AIR** | Asset Information Requirements — what's needed for the operational phase |
| **PIR** | Project Information Requirements — what's needed during the delivery phase |
| **OIR** | Organizational Information Requirements — strategic requirements of the appointing party |
| **PIM** | Project Information Model — the information model during the delivery phase |
| **AIM** | Asset Information Model — the information model during the operational phase |
