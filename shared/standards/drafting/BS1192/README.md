# BS 1192:2007+A2:2016 — Drafting Standards Layer

UK information management standard. Covers Common Data Environment (CDE), file naming, layer naming, revision conventions, and status/suitability codes.

## Files in this folder

| File | Purpose |
|---|---|
| `meta.json` | Publication metadata, edition, amendments, scope |
| `layer-naming.json` | §6 — Role-Discipline-Number-Type layer naming convention |
| `status-codes.json` | §5 — Suitability status codes (S0-S7) |
| `file-naming.json` | §7 — Project-Originator-Volume-Level-Type-Role-Number file naming |
| `revision-conventions.json` | §8 — P/C-regime alphanumeric revision sequencing |
| `project-codes.json` | §7.4 — Discipline role codes + originator conventions |
| `cad-layers.json` | Skills-layer abstract `layout_group` → BS 1192 concrete CAD layer code mapping (consumed by SLD v1.5+ runtime renderer) |
| `terminology.md` | Key terms: WIP/Shared/Published/Archived/CDE/EIR/AIR/PIR/OIR |

## Cross-references

- **ISO 19650** — successor international standard (BS 1192 largely superseded in UK since 2018, but BS 1192:2007+A2:2016 remains the canonical layer-naming reference)
- **BS 5536** — UK reading of ISO 128 line conventions for technical drawings
- **ISO 13567** — referenced by BS 1192 §6.3 for layer naming alignment
