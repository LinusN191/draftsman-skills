# ISO 129-1:2018 — Dimensions and Tolerances

International standard for dimensioning and tolerance presentation on technical drawings.

## Files in this folder

| File | Purpose |
|---|---|
| `meta.json` | Publication metadata, edition, scope |
| `dimensioning-rules.json` | §5 + §6 — chained/parallel/running methods + leader conventions |
| `tolerance-conventions.json` | §7 + ISO 2768 — numerical, class (f/m/c/v), and geometric tolerances |
| `symbols.json` | §4 — 10 dimensioning symbols (Ø, R, SR, □, etc.) |

## Engine-lookup keys

- `symbols.engine_lookup.by_name["diameter"]` → "Ø"
- `dimensioning-rules.engine_lookup.methods_list` → `["chained", "parallel", "running"]`

## Applies to jurisdictions

- **GB / KE / INT / EU / US** — primary standard (widely adopted internationally including US for technical product documentation)
