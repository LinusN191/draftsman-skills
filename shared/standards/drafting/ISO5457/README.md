# ISO 5457:1999/A1:2010 — Sizes and Layout of Drawing Sheets

International standard for technical drawing sheet geometry. Covers A0-A4 sheet sizes (cut + trimmed dimensions), drawing area, title block region, and grid references.

## Files in this folder

| File | Purpose |
|---|---|
| `meta.json` | Publication metadata, edition, amendments, scope |
| `sheet-sizes.json` | Clause 3 + Tables 1-3 — cut/trimmed dimensions A0-A4 in mm; engine-lookupable by sheet_size key |
| `drawing-area-rules.json` | Clause 5 — margins (25mm left filing edge, 10mm other sides), grid references per sheet size |
| `cross-references.json` | Cross-refs to ISO 7200 (title block), ISO 9431 (folds), ISO 5455 (scales) |

## Cross-references

- **ISO 7200** — title block data fields
- **ISO 9431** — drawing fold procedures
- **ISO 5455** — standard drawing scales

## Applies to jurisdictions

- **GB / KE / INT / EU** — primary standard (ISO A-series widely adopted)
- **US** — typically uses ANSI sizes (Arch_A through Arch_E) per AIA conventions; ISO sizes occasionally used for international projects
