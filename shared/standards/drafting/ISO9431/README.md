# ISO 9431:1990 — Construction Drawings: Folding Procedures

International standard for folding technical drawings to A4 size for archival filing. Title block remains visible on the front face after folding.

## Files in this folder

| File | Purpose |
|---|---|
| `meta.json` | Publication metadata, edition, scope |
| `drawing-fold-procedures.json` | §6 — fold sequences A0/A1/A2/A3 → A4 + fold count per sheet size |

## Engine-lookup keys

- `engine_lookup.fold_count_to_a4["A0"]` → 4
- `engine_lookup.fold_count_to_a4["A1"]` → 3

## Cross-references

- **ISO 5457** — sheet sizes that are folded
- **ISO 7200** — title block (must remain visible after folding)

## Applies to jurisdictions

- **GB / KE / INT / EU** — primary standard
- **US** — rarely used (US printing typically uses ANSI sheet sizes which fold differently)
