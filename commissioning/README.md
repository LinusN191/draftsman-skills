# commissioning

Cross-discipline testing, commissioning, and certification skills for MEP systems. Covers the BS 7671 Part 6 inspection-test path (EIC/EICR), CIBSE/BSRIA commissioning specifications (BG 6 + Codes A–W), and witness-test scheduling for client/consultant sign-off.

Each commissioning skill consumes design intent from one or more upstream design skills (electrical/mechanical/plumbing) and produces a structured handover artifact — a certificate, plan, or witness-test record — that gates an installation's legal usability.
